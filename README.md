# Aadhaar Enrolment Analytics with Machine Learning

## Overview
This project analyses UIDAI Aadhaar enrolment data using data analytics and machine learning techniques.
It includes enrolment trend analysis, demographic insights, forecasting, and regional clustering.

## Datasets
UIDAI Aadhaar enrolment datasets (provided for the hackathon):
- api_data_aadhar_enrolment_0_500000.csv
- api_data_aadhar_enrolment_500000_1000000.csv
- api_data_aadhar_enrolment_1000000_1006029.csv

## Methodology
- Data cleaning and preprocessing
- Exploratory data analysis
- Feature engineering
- Time-series forecasting using Prophet
- Regional clustering using K-Means
